1. Introducción

Estos términos y condiciones regulan el uso de este portafolio profesional y cualquier interacción relacionada. Al acceder a este sitio web, aceptas cumplir con las normas aquí establecidas. Si no estás de acuerdo con alguno de estos términos, se recomienda no utilizar este sitio.

2. Propiedad intelectual

Todo el contenido de este sitio, incluyendo textos, imágenes, código fuente y diseño, es propiedad del titular del portafolio, salvo que se indique lo contrario. El uso indebido del contenido puede derivar en acciones legales.

3. Uso del contenido

El contenido de este portafolio está destinado exclusivamente para fines profesionales y de revisión académica. No está permitido copiar, modificar, distribuir o utilizarlo con fines comerciales sin la autorización expresa del titular.

4. Responsabilidad del usuario

El usuario se compromete a utilizar este sitio de manera responsable y ética. Cualquier uso malintencionado, como la alteración de contenido, intentos de acceso no autorizado o distribución de información falsa, está estrictamente prohibido.

5. Contacto profesional

El formulario de contacto debe usarse exclusivamente para consultas relacionadas con oportunidades laborales, colaboraciones profesionales o proyectos de interés mutuo. No se permite su uso para envío de publicidad, spam o cualquier otro fin no autorizado.

6. Confidencialidad y protección de datos

Este sitio no almacena ni recopila datos personales sin el consentimiento del usuario. Cualquier información compartida a través del formulario de contacto será tratada con la debida confidencialidad y no se divulgará a terceros sin autorización.

7. Enlaces a terceros

Este portafolio puede contener enlaces a sitios web externos que no están bajo el control del titular. No se asume responsabilidad por el contenido, prácticas de privacidad ni políticas de dichos sitios.

8. Disponibilidad del sitio

El titular del portafolio se reserva el derecho de modificar, suspender o eliminar el sitio web sin previo aviso. No se garantiza la disponibilidad continua del servicio.

9. Exención de responsabilidad

El titular no se hace responsable de errores, omisiones o inexactitudes en el contenido del portafolio. El usuario asume la responsabilidad de verificar la información y hacer un uso adecuado de ella.

10. Modificaciones

Estos términos pueden actualizarse en cualquier momento sin previo aviso. Se recomienda a los usuarios revisarlos periódicamente para estar informados sobre posibles cambios.
